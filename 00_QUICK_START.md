# Quick Start

## Outcome

load Pydantic AI agent runtime context, tool-call boundaries, state handoff checks, tracing notes, and eval guidance into your AI host before building agent apps

## Two-Minute Path

1. Read `AGENTS.md`.
2. Copy the relevant host instruction into your AI coding host.
3. Run `06_EVALS/smoke_check.md`.
4. If the check fails, open `03_PITFALL_LOG.md`.
